# 💰 Finsight Analytics — Personal Finance Tracker

A clean, browser-based personal finance tracker with dashboard, analytics, transaction management, and smart insights. Built with HTML, CSS, and JavaScript — no backend required.

---

## 🚀 Live Pages

| Page | File | Description |
|------|------|-------------|
| 🏠 Dashboard | [index.html](index.html) | Income, Expense & Savings overview with charts |
| 📊 Analytics | [Pages/analytics.html](Pages/analytics.html) | Visual breakdown of spending patterns |
| ➕ Add Transaction | [Pages/add-transaction.html](Pages/add-transaction.html) | Log new income or expense entries |
| 📋 Transactions | [Pages/transaction.html](Pages/transaction.html) | Full transaction history list |
| 📄 Reports | [Pages/reports.html](Pages/reports.html) | Monthly and category-wise reports |
| 👤 Profile | [Pages/profile.html](Pages/profile.html) | User profile settings |

---

## ✨ Features

- 📊 **Interactive Charts** — Toggle between Bar and Pie chart views (powered by Chart.js)
- 💡 **Smart Insights** — Automatic financial tips based on your data
- ➕ **Add Transactions** — Log income and expense entries easily
- 📋 **Transaction History** — View all past transactions in one place
- 📄 **Reports** — Month-wise and category-wise financial reports
- 👤 **User Profile** — Personalized profile management
- 🎨 **Animated Values** — Smooth number animations on the dashboard
- 📱 **Responsive UI** — Works on desktop and mobile browsers
